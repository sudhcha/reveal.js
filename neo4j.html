NoSQL Space

  • Key-Value Stores
    • Best-Known Technologies:
      • Redis
      • Memcached
    • Typical Use Cases
      • Caches
      • Simple domain with fast read access
      • Massively concurrent systems

NoSQL Space

  • Column-Family Stores
    • Best-Known Technologies:
      • Cassandra
      • Google Big Table
    • Typical Use Cases
      • Write on a big scale
      • Co-located data access (for reading and writing)

NoSQL Space

  • Document Oriented Databases
    • Best-Known Technologies:
      • MongoDB
      • Couchbase
    • Typical Use Cases
      • When domain model is document by nature
      • To simplify development using natural document data structures
      • Highly scalable systems (although on the lower level then the key-value and column-family stores)

NoSQL Space

  • Graph databases
    • Best-Known Technologies:
      • Neo4j
      • AllegroGraph
      • OrientDB
    • Typical Use Cases
      • With inteconnected data
      • Domain can be represented with nodes and relationships naturally
      • Social Networks
      • Recommendation engines
      • Access Control Lists

Neo4j Data Storage

  • Nodes => vertices
